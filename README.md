# SSL证书查询助手 - Chrome 插件

一键查询当前网站的SSL证书信息，快速了解网站安全状态。

## ✨ 功能特性

- 🔒 **安全状态检测** — 一眼看出网站是否使用 HTTPS
- 📋 **证书详情** — 显示颁发机构、有效期、序列号等信息
- ⏰ **到期提醒** — 显示证书剩余有效期，临近过期会变色提醒
- 🔍 **智能查询** — 通过 crt.sh API 获取详细证书信息
- 🎨 **精美界面** — 深色主题，视觉舒适

## 🚀 安装方法

1. 打开 Chrome，访问 `chrome://extensions/`
2. 开启右上角的**开发者模式**
3. 点击**加载已解压的扩展程序**
4. 选择本项目文件夹

## 📖 使用方法

1. 访问任意网站
2. 点击工具栏中的插件图标
3. 查看当前网站的 SSL 证书信息

## 📊 显示信息

- **连接状态** — 安全/不安全
- **协议类型** — HTTPS/HTTP
- **证书颁发机构** — CA 机构名称
- **颁发对象** — 证书对应的域名
- **生效时间** — 证书开始生效日期
- **到期时间** — 证书过期日期
- **剩余有效期** — 距离过期的天数
- **证书序列号** — 唯一标识

## 🛠️ 技术栈

- Manifest V3
- Chrome Extensions API
- crt.sh Certificate Transparency API
- 纯 JavaScript（无框架）

## 📝 注意事项

- HTTP 网站会显示"连接不安全"
- 部分证书信息可能需要几秒钟才能加载
- Chrome 内部页面（chrome://）无法查询
